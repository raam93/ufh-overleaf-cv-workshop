Creating Resumes and Cover Letters with Overleaf
=================================================

Resources for the **Creating Resumes and Cover Letters with Overleaf** workshop at University Family Housing, University of Toronto.

Files
-----

* ``index.html`` — workshop home
* ``cv_academic.tex`` — LaTeX CV template for PhD students and researchers
* ``cv_professional.tex`` — LaTeX CV template for working professionals
* ``cover_letter_academic.tex`` — cover letter template using the letter document class
* ``cover_letter_professional.tex`` — cover letter template with manual layout using the article class
